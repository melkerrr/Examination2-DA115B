Usage
=====

.. _installation:

Installation
------------

To use Examination2-DA115B, install it using Git Clone.

.. code-block:: console

.. autofunction::
   (.venv) $ git clone github.com/melkerrr/Examination2-DA115B.git


Starting the Game
----------------

Use the cd (Change directory) command to navigate to the directory where your Python script is located.
Folder called ``cd Examination2-DA115B``:

.. autofunction:: cd Examination2-DA115B

To Run the python script use "python main.py" to start the game. 

.. autoexception:: python main.py

For example:
>>> git clone github.com/melkerrr/Examination2-DA115B.git
>>> cd Examination23\Examination2-DA115B
>>> python main.py
